#  Weather Information Application (React)

##  Description

This project is a **React-based weather application** that fetches real-time and historical weather data using the **OpenWeatherMap API**. It displays weather information such as temperature, humidity, and conditions, along with graphical visualization using charts.

---

##  Objective

To create a React service that retrieves weather data from an external API and presents both current and historical weather information in a user-friendly interface using charts.

---

##  Technologies Used

* ReactJS
* JavaScript (ES6)
* HTML & CSS
* OpenWeatherMap API
* Chart.js (for data visualization)

---

##  Components / Modules

1. **App Component**

   * Main container managing overall structure
2. **Weather Service**

   * Handles API requests to fetch weather data
3. **Search Component**

   * Allows users to enter city name
4. **Weather Display Component**

   * Shows current weather details
5. **Chart Component**

   * Displays historical weather data using graphs

---

##  Features

* Fetch current weather data
* Fetch historical weather data
* Display temperature, humidity, and weather conditions
* Visualize data using charts
* User-friendly interface

---

##  API Used

* OpenWeatherMap API

  * Current Weather API
  * Historical Weather API

---

##  How to Run the Project

1. Install Node.js
2. Create React App:

   ```
   npx create-react-app weather-app
   ```
3. Navigate to project folder:

   ```
   cd weather-app
   ```
4. Install dependencies:

   ```
   npm install axios chart.js react-chartjs-2
   ```
5. Add your OpenWeatherMap API key in the project
6. Start the application:

   ```
   npm start
   ```
7. Open browser:

   ```
   http://localhost:3000
   ```

##  Output

* Search bar to enter city name
* Displays current weather details
